# Frontend Development Learning Repository

This repository is a structured React learning journey built through step-by-step notes and hands-on mini projects.

## Why This Repo Exists

This repo exists to document practical frontend learning from basics to component-based app building.

It helps you:
- Keep topic notes and project work in one place.
- Practice by building small projects after each concept.
- Revisit concepts quickly using both theory files and code examples.

## What You Will Learn

By following this repository, you will learn:
- React fundamentals and project setup using Vite.
- Creating and organizing React components.
- Using Bootstrap with React layouts.
- Building beginner apps like a Todo App and a Bharat Clock.
- Writing basic project documentation and tracking limitations for future improvements.

### Notes
- Markdown files at the root explain each learning topic.
- Numbered folders contain concept-specific practice apps.
- Projects folder contains mini apps with dedicated README files.

## Who Is This For?

This repository is ideal for:
- Beginners starting with React.
- Students building frontend practice projects.
- Anyone who wants a clean, incremental learning path with examples.
